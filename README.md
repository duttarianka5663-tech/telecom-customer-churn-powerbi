## Project Title
Telecom Customer Churn Analysis - Power BI Dashboard

## Objective
To analyze customer churn patterns in a telecom company 
and identify key factors driving customer attrition.

## Tools Used
Power BI, DAX, Data Visualization

## Key Findings
- Overall churn rate identified at 26.54% across 7,043 customers
- Month-to-month contract customers showed highest churn tendency
- New customers (0-12 months tenure) had significantly higher churn
- Fiber optic internet service users showed higher churn rate

## Dataset
Telecom customer dataset with demographics, 
contract, and service usage data
